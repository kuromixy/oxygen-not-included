2. Create `src/components/ThemeToggle.astro`:
   - Button with id "theme-toggle", aria-label "Toggle dark mode"
   - Styled: `p-2 rounded-lg text-oni-text-secondary hover:text-oni-teal hover:bg-oni-bg-elevated transition-colors`
   - Sun SVG icon (id="icon-sun", hidden class) -- shown when dark mode is active (click to switch to light)
   - Moon SVG icon (id="icon-moon", hidden class) -- shown when light mode is active (click to switch to dark)
   - Inline script:
     - `updateIcons()` function: reads `document.documentElement.classList.contains('dark')`, shows sun if dark, moon if light
     - Click handler: toggles `dark` class on `document.documentElement`, saves to localStorage as 'theme' key ('dark'/'light'), calls updateIcons()
     - Call updateIcons() on load

3. Create `src/layouts/BaseLayout.astro`:
   - Accept `title` prop (string)
   - Full HTML document: `<!DOCTYPE html>`, `<html lang="en" class="dark">` (default dark)
   - `<head>`: charset, viewport, title prop, favicon
   - CRITICAL: Dark mode inline script BEFORE any stylesheet import -- prevents FOUC:
     ```
     <script is:inline>
       const theme = (() => {
         if (typeof localStorage !== 'undefined' && localStorage.getItem('theme')) {
           return localStorage.getItem('theme');
         }
         if (window.matchMedia('(prefers-color-scheme: dark)').matches) {
           return 'dark';
         }
         return 'light';
       })();
       if (theme === 'dark') {
         document.documentElement.classList.add('dark');
       } else {
         document.documentElement.classList.remove('dark');
       }
     </script>
     ```
   - AFTER the inline script: `import '../styles/global.css';`
   - `<body>` with `bg-oni-bg-deep text-oni-text-primary min-h-screen` (dark mode defaults), plus light mode variants: `dark:bg-oni-bg-deep dark:text-oni-text-primary bg-slate-50 text-slate-900` -- but since dark is default and class-based, use: `class="bg-oni-bg-deep text-oni-text-primary min-h-screen"` for dark, and override in global.css for light via removing .dark class
   - Include Navbar component at top of body
   - Main content area: `<main class="max-w-6xl mx-auto px-4 py-8">` with `<slot />` for page content
   - Simple footer with "ONI Reference -- Fan-made quick reference" text, muted style
