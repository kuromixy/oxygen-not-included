# ONI Fanpage

## What This Is

A quick-reference fanpage for Oxygen Not Included — a playful, ONI-themed cheat sheet site where players can look up optimal build designs, key stats, and room dimensions at a glance while playing. Deployed as a static site on GitHub Pages.

## Core Value

Players can instantly find the exact number, dimension, or layout they need mid-game without wading through wikis or forum posts.

### Out of Scope

- Full wiki / exhaustive database — this is a cheat sheet, not a wiki
- Community features (comments, submissions, forums) — static site, no backend
- News/patch notes tracking — focus is on reference data
- Mobile app — web only
- User accounts or authentication — purely static content

## Context

- Oxygen Not Included is a colony survival simulation by Klei Entertainment
- The game has deep mechanics around ranching, farming, power systems, plumbing, and base design
- Players frequently need to look up optimal dimensions and ratios mid-game
- Existing resources (wiki, forums) are comprehensive but slow to navigate for quick lookups
- GitHub Pages provides free static hosting with simple deployment from a repo
- Site will use placeholder images initially; real screenshots/diagrams added over time
- Tech stack to be determined by research (plain HTML/CSS/JS vs static site generator)

## Constraints

- **Hosting**: GitHub Pages — static files only, no server-side processing
- **Content**: Reference data must be accurate to current game version (DLC included)
- **Images**: Placeholder-first approach, structure must support easy image replacement later
